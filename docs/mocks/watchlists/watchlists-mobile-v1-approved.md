# Watchlists Mobile V1 (Approved)

Status: APPROVED
Owner: Product Owner
Date: 2026-06-02

## Layout

- Watchlist tabs σε μία γραμμή (single-line tabs)
- Horizontal scroll χωρίς visible scrollbar (hidden scrollbar)
- + = Add Instrument στην τρέχουσα watchlist (adds to current list)
- Table/Cards ΔΕΝ εμφανίζεται στη βασική οθόνη (no table/cards on main view)
- View selection μέσα στο menu των 3 τελειών (view selection inside three-dots menu)

## Cards

- Portfolio-card style
- Χωρίς position metrics (no position metrics)
- Momentum bar
- Risk bar
- 1x1 view
- 2x2 grid view

## Long Press

Ticker actions:

- Delete from Watchlist
- View Chart
- Add to List
- AI Coach

## Three Dots Menu

- New Watchlist
- Rename Watchlist
- Manage Watchlists
- Manage Columns
- Cards 1x1
- Cards 2x2

## Manage Columns

Single unified list

Each row:
- Column Name
- On/Off Toggle
- 6-dot drag handle

Rules:
- Instrument locked
- Smooth drag reorder
- No arrows
- No second ordering panel
- Hidden scrollbar

## Adaptive

Target breakpoints:
- 390px (small mobile)
- 430px (large mobile)
- Tablet
- Desktop

Design approach: Mobile-first design.
